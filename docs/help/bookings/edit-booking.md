# Edit a Booking

You can edit a booking's details after it has been created. Only owners and admins can edit bookings.

## Open the Booking

1. Go to **Bookings** in the sidebar.
2. Find the booking in the list — use the search or filter by status if needed.
3. Click the booking number to open its detail page.
4. Click **Edit** in the top-right button group.

## Editable Fields

The edit form lets you change:

- **Customer** — reassign the booking to a different customer.
- **Trip** — move the booking to a different scheduled trip.
- **Number of Participants** — adjust the headcount. DiveStreams will validate that the new number fits within the destination trip's available capacity.
- **Participant Details** — update participant names.
- **Tank & Gas Selection** — change or add tank/gas details per participant.
- **Equipment Rental** — add or remove rental items.
- **Special Requests** — update any customer-facing notes.
- **Internal Notes** — update staff-only notes.
- **Booking Source** — correct the source if it was entered incorrectly.

## Save Changes

Click **Save Changes** after making your edits. DiveStreams updates the booking and recalculates the total if the trip price or participant count changed.

> **Tip:** If you need to move a booking to a different date, use the **Trip** dropdown on the edit form to select the new trip.

## What You Cannot Edit

- The booking number (auto-assigned and immutable)
- Payment records (payments are added through the [payment workflow](./manage-payments.md))
- The booking status (use the action buttons on the detail page to transition status)

## Printing a Booking

On the booking detail page, click **Print Booking** under Quick Actions in the sidebar. A print-ready page opens in a new window with all trip, customer, and payment details.
