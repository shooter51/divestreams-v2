# Add a Boat

**Boats** represent the vessels in your fleet. Once added, you can assign boats to trips so the trip manifest shows which vessel is being used.

## Steps

1. Go to **Boats** in the sidebar.
2. Click **Add Boat** in the top right.

### Basic Info

| Field | Required | Notes |
|-------|----------|-------|
| Boat Name | Yes | e.g. "Sea Eagle", "Manta 1" |
| Boat Type | No | Dive Boat, Speed Boat, Catamaran, Yacht, RIB, Other |
| Passenger Capacity | Yes | Maximum number of passengers (1–100) |
| Description | No | Description shown on public site |

### Registration

- **Registration Number** — your vessel's official registration or licence number (e.g. "MV-1234-TH"). This is for your records only and is not publicly displayed.

### Amenities & Features

Click amenity buttons to add them to your boat's feature list. Available amenities include:
- Dive platform
- Sun deck
- Toilet
- Freshwater shower
- Camera station
- Storage lockers
- Shade cover
- First aid kit
- Sound system
- BBQ grill

Selected amenities appear as removable chips above the selection buttons. Click the × on a chip to remove it. Amenities are shown on your public site's boat information.

### Images (Optional)

Upload up to 5 images (JPEG, PNG, WebP, or GIF; max 10 MB each). You can add more after creation from the boat edit page.

### Status

Check **Active** to make the boat available to be assigned to trips. Uncheck to remove it from the trip scheduling form without deleting the record.

## Save

Click **Add Boat**. You are redirected to the boat edit page where you can add more images via the image manager.

## After Adding a Boat

The boat immediately appears in the **Select Boat** dropdown when scheduling a new trip.
