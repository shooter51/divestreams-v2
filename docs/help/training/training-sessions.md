# Training Sessions

A **Training Session** is a scheduled instance of a Course — a specific date, location, and instructor for a group of students. You must schedule at least one Session before you can enrol students.

## Schedule a Session

1. Go to **Training** in the sidebar.
2. Click **Sessions** in the training sub-navigation.
3. Click **New Session** in the top right.

### Select Course

Use the **Select Course** dropdown to choose which course this session is for. All active courses appear in the list with their agency (e.g. "PADI Open Water Diver (PADI)").

Once selected, a summary panel shows the course's duration, price, and default max students.

### Date & Time

| Field | Required | Notes |
|-------|----------|-------|
| Start Date | Yes | The first day of the session |
| End Date | No | Leave blank if the session is a single day |
| Start Time | No | When the session begins each day |

### Location

- **Location/Venue** — e.g. "Shop classroom", "City Aquatics Pool", "Tulamben Beach"
- **Meeting Point** — specific instructions for where students should gather (e.g. "Main entrance, ground floor")

### Instructor & Capacity

- **Instructor Name** — free text, e.g. "Maria Santos" or "Instructor Team"
- **Max Students** — override the course default for this session. Leave blank to use the course's default.

### Pricing

- **Price Override** — set a different price for this session. Leave blank to use the course's default price.

### Session Type

Select the type of training taking place:
- Classroom
- Pool
- Confined Water
- Open Water
- Exam
- Other
- Not Specified

### Internal Notes

Any notes for your staff about this session.

## Save

Click **Create Session**. You are redirected to the Sessions list.

## Managing a Session

From the session detail page you can:
- View and manage enrolled students
- Add new enrollments
- Update the session status (Scheduled → In Progress → Completed → Cancelled)
- See payment status for each enrollment

## Session Status

| Status | Meaning |
|--------|---------|
| Scheduled | Session is upcoming |
| In Progress | Session is currently running |
| Completed | Session has finished |
| Cancelled | Session was cancelled; no new enrollments allowed |
