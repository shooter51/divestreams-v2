# Edit a Tour

You can edit any tour's details at any time. Changes to a tour's price or capacity apply to **future** trips created from it — they do not retroactively change existing scheduled trips.

## Steps

1. Go to **Tours** in the sidebar.
2. Click the tour name to open its detail page.
3. Click **Edit** in the top right.
4. Make your changes.
5. Click **Save Changes**.

## What You Can Change

All fields from the creation form are editable:

- Tour Name and Description
- Tour Type and Duration
- Price and Currency
- Max and Min Participants
- Inclusions, Exclusions, Requirements
- Minimum Certification and Minimum Age
- Require tank & gas selection toggle
- Active status

## Updating Images

On the tour detail page (not the edit form), you will find an **Images** section with the image manager. From there you can:
- Upload additional images (up to 5 total)
- Drag to reorder images
- Delete existing images
- Set a different image as the primary photo

## Deactivating a Tour

If you want to stop scheduling new trips from a tour without deleting it:

1. Open the tour edit form.
2. Uncheck **Active**.
3. Save.

Inactive tours are hidden from the "New Trip" tour selector, so you cannot accidentally schedule a trip from them.

## Effect on Existing Trips and Bookings

Editing a tour does **not** change:
- Existing scheduled trips (they keep their original price and capacity)
- Existing bookings (prices are locked at booking time)

If you need to adjust the price for a specific trip only, use the **Price Override** field when scheduling the trip. See [Create a Trip](../trips/create-trip.md).
