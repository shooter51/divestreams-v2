# Set Up Your First Tour

In DiveStreams, a **Tour** is a reusable template that describes a dive experience — its type, price, capacity, and what's included. Every **Trip** (a scheduled date/time instance) is based on a Tour. You need at least one Tour before you can schedule Trips or take Bookings.

## Step 1: Create the Tour

1. Click **Tours** in the sidebar.
2. Click **New Tour** in the top right.
3. Fill in **Tour Name** — for example, "Morning Two-Tank Reef Dive". This name appears on your public booking page.
4. Optionally add a **Description** that appears on your public site.
5. Select a **Tour Type**:
   - Single Dive
   - Multi Dive
   - Course
   - Snorkel
   - Night Dive
   - Other
6. Enter the **Duration** in minutes (e.g. `120` for a two-hour trip).

## Step 2: Set Pricing and Capacity

1. Enter the **Price** per person (minimum $1).
2. Select the **Currency** (USD, EUR, GBP, AUD, THB, IDR, MXN).
3. Set **Max Participants** — the maximum number of divers per trip instance.
4. Optionally set **Min Participants** — the minimum needed for the trip to run.

## Step 3: Configure What's Included

Check any inclusions that apply:

- **Equipment Rental** — tanks, BCDs, regulators, wetsuits included in the price
- **Meals/Snacks** — food is included
- **Transport** — transfers are included
- **Require tank & gas selection** — if enabled, customers must select tank type and gas mix when booking

You can also add comma-separated lists for **Additional Inclusions**, **Exclusions**, and **Other Requirements**.

## Step 4: Set Requirements

- **Minimum Certification** — choose None, Open Water, Advanced Open Water, Rescue Diver, or Divemaster
- **Minimum Age** — enter a number if there is an age restriction

## Step 5: Upload Images (Optional)

Upload up to 5 images (JPEG, PNG, WebP, GIF; max 10 MB each). The first image becomes the primary photo shown on your public site.

## Step 6: Save

Click **Create Tour**. DiveStreams saves the tour and redirects you to the tour detail page where you can add more images or edit details.

## Next: Schedule a Trip

A Tour by itself doesn't appear as a bookable date — you need to schedule at least one Trip from it. See [Create a Trip](../trips/create-trip.md).
