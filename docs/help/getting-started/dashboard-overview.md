# Dashboard Overview

The Dashboard is the first page you see after logging in. It gives you a real-time summary of your operation and quick access to every section of DiveStreams.

## Stats Cards

At the top of the Dashboard you see four at-a-glance metrics:

- **Today's Bookings** — the number of new bookings created today.
- **Total Customers** — the total number of customers in your account.
- **Scheduled Trips** — trips with status "scheduled" (not yet completed or cancelled).
- **Week's Revenue** — total revenue from bookings created this week.

## Upcoming Trips

Below the stats, the Dashboard shows a list of your upcoming scheduled trips. Each row shows:

- Tour name
- Date and start time
- Number of bookings / capacity
- Status badge

Click any trip to open the trip detail page.

## Recent Bookings

The Dashboard also shows your most recent bookings. Each row shows the booking number, customer name, trip, and status. Click a booking to open the full booking detail.

## Sidebar Navigation

The left sidebar is your main navigation. It contains:

| Section | What you do there |
|---------|------------------|
| **Dashboard** | Home page with stats and upcoming trips |
| **Calendar** | Visual calendar view of all trips |
| **Bookings** | All bookings — create, view, manage payments |
| **Trips** | Schedule and manage individual trip instances |
| **Tours** | Define the reusable tours that trips are based on |
| **Customers** | Customer database |
| **Equipment** | Rental and owned equipment inventory |
| **Boats** | Your fleet |
| **Training** | Courses, sessions, and student enrollments |
| **Gallery** | Photo albums |
| **POS** | Point of Sale — retail, rentals, quick bookings |
| **Products** | Products for sale in the POS |
| **Reports** | Export booking data |
| **Discounts** | Discount codes |
| **Settings** | Account, team, integrations, public site |

## Plan Limits

If you approach a plan limit (e.g. number of tours or customers), a warning banner appears on the relevant page. Upgrade your plan from **Settings** → **Billing & Subscription** to increase limits.
