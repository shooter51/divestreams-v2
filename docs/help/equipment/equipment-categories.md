# Equipment Categories

Every equipment item belongs to a category. Categories are used to filter the equipment list and to organise items in the POS rental panel.

## Available Categories

| Category | Description |
|----------|-------------|
| **BCD** | Buoyancy Control Devices |
| **Regulator** | First and second stages, octopus |
| **Wetsuit** | Wetsuits, drysuits, rashguards |
| **Mask** | Masks, snorkels, fins (face equipment) |
| **Fins** | Fins / flippers |
| **Tank** | Dive cylinders / scuba tanks |
| **Computer** | Dive computers, dive lights |
| **Other** | Any equipment that does not fit the above |

## Tanks as a Special Category

When you select **Tank** as the category, an additional field appears: **Gas Type**. This records what the tank is filled with:
- Air
- Nitrox 32%
- Nitrox 36%
- Trimix
- Oxygen

This gas type is separate from the **Tank Types** used in the booking tank selection feature. The gas type on an equipment record describes the physical tank's current fill. Tank Types for booking selection are managed in the tank setup section.

## Filtering by Category

On the Equipment list page, use the category filter buttons at the top to show only items of a specific category. This is useful when checking how many wetsuits or BCDs you have available.

## Category in the POS

In the Point of Sale, the **Rentals** tab groups available rental equipment by category. Customers can see items organised by type, making it easy to find and select what they need.
