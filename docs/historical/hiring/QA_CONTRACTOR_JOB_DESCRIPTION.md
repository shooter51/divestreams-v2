## About DiveStreams

DiveStreams is a multi-tenant SaaS platform for dive shop and tour operator management. The platform helps dive businesses manage bookings, customers, equipment, training courses, and more.

## Role Overview

We are seeking a detail-oriented QA Contractor to perform comprehensive manual testing of our web application. You will execute test cases documented in Jira, report bugs, verify fixes, and ensure the platform meets quality standards before production release.

## Responsibilities

- Execute **543 pre-defined test cases** in Jira (KAN-49 through KAN-588)
- Document test results directly in Jira issues
- Report new bugs discovered during testing with detailed reproduction steps
- Verify bug fixes and update issue status
- Test across multiple browsers (Chrome, Firefox, Safari, Edge)
- Test responsive design on tablet and mobile viewports
- Provide feedback on UX issues and improvements
- Participate in brief async standups via Slack/email

## Required Skills

- **2+ years** of manual QA testing experience
- Experience with **Jira** for test case management
- Understanding of web application testing methodologies
- Strong attention to detail and documentation skills
- Excellent written communication in English
- Self-motivated and able to work independently
- Familiarity with browser developer tools

## Nice to Have

- Experience testing SaaS/B2B applications
- Experience with multi-tenant applications
- Familiarity with Playwright or other E2E testing tools
- Experience in hospitality, tourism, or booking systems
- Basic understanding of REST APIs
