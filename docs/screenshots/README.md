## Screenshot Guidelines

When capturing screenshots:

1. **Resolution**: Use 1440px width for consistency
2. **Browser**: Use Chrome with no extensions visible
3. **Data**: Use demo data that looks realistic
4. **Privacy**: Ensure no real customer data is visible
5. **Format**: Save as PNG for quality
6. **Naming**: Use kebab-case matching the reference names above

## Capture Steps

To capture screenshots:

1. Navigate to https://demo.staging.divestreams.com
2. Log in with demo credentials
3. Navigate to each section
4. Use browser dev tools to set viewport to 1440px wide
5. Capture the relevant area
6. Save with the corresponding filename
